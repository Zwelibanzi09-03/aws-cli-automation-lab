# AWS CLI Automation Lab

## Overview
This project demonstrates how to use the AWS Command Line Interface (CLI) to collect and automate infrastructure data across AWS services.

The goal was to build a simple inventory system that retrieves information about:
- Amazon S3 buckets  
- Amazon EC2 instances  
- AWS IAM users  

and automate that process using reusable scripts.

---

## Project Goal
To understand how cloud engineers interact with AWS programmatically by:
- Using CLI instead of the AWS Console  
- Automating repetitive cloud tasks  
- Creating reusable scripts  

---

## Technologies Used
- AWS CLI  
- PowerShell (Windows)  
- Bash (Linux)  
- Git & GitHub  

### AWS Services
- Amazon S3  
- Amazon EC2
